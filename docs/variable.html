Variables difusas¶

class fuzzy_expert.variable.FuzzyVariable(universe_range, terms=None, step=0.1)[fuente]¶

Bases: object

Crea una variable difusa.

Parámetros

  • unverse_range – Límites del universo de discurso.

  • terms – Diccionario donde cada término es la clave del diccionario y los valores son la función de pertenencia.

  • step – Valor que controla la resolución de la representación discreta del universo.

>>> from fuzzy_expert.variable import FuzzyVariable
>>> v = FuzzyVariable(
...     universe_range=(150, 200),
...     terms={
...         "High": [(175, 0), (180, 0.2), (185, 0.7), (190, 1)],
...         "Low": [(155, 1), (160, 0.8), (165, 0.5), (170, 0.2), (175, 0)],
...     },
... )
>>> v.plot()

get_modified_membeship(term, modifiers=None)[fuente]¶

Devuelve los valores modificados de pertenencia para el término.

Parámetros

  • term – Nombre del conjunto difuso.

  • modifiers – Lista de modificadores.

>>> import matplotlib.pyplot as plt
>>> from fuzzy_expert.variable import FuzzyVariable
>>> v = FuzzyVariable(
...     universe_range=(150, 200),
...     terms={
...         "High": [(175, 0), (180, 0.2), (185, 0.7), (190, 1)],
...         "Low": [(155, 1), (160, 0.8), (165, 0.5), (170, 0.2), (175, 0)],
...     },
... )
>>> y = v.get_modified_membeship('High' ,['extremely'])
>>> _ = plt.plot(v.universe, v['High'], label='High')
>>> _ = plt.plot(v.universe, y, label='extremely High')
>>> _ = plt.legend()
>>> plt.show()
