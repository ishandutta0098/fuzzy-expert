Método de Inferencia¶

class fuzzy_expert.inference.DecompositionalInference(and_operator, or_operator, implication_operator, composition_operator, production_link, defuzzification_operator)[fuente]¶

Bases: object

Método descomposicional de inferencia.

Parámetros

  • and_operator – Método del operador AND para combinar las composiciones de proposiciones en una premisa de regla difusa, especificada como una de las siguientes: * «min». * «prod». * «bunded_prod». * «drastic_prod».

  • or_operator – Método del operador OR para combinar las composiciones de proposiciones en una premisa de regla difusa, especificada como una de las siguientes: * «max». * «prob_or» * «bounded_sum». * «drastic_sum».

  • implication_operator – Método para calcular las composiciones de proposiciones en una premisa de regla difusa, especificado como uno de los siguientes: * «Ra». * «Rm». * «Rc». * «Rb». * «Rs». * «Rg». * «Rsg». * «Rgs». * «Rgg». * «Rss».

  • production_link – Método para agregar las consecuencias de las reglas difusas, especificado como uno de los siguientes: * «min». * «prod». * «bunded_prod». * «drastic_prod». * «max». * «prob_or». * «bounded_sum». * «drastic_sum».

  • defuzzification_operator – Método para defuzzificar la función de pertenencia resultante, especificado como uno de los siguientes: * «cog»: Centro de gravedad. * «boa»: Bisector del área. * «mom»: Media de los valores para los cuales la función de pertenencia es máxima. * «lom»: Máximo de los valores para los cuales la función de pertenencia es máxima. * «som»: Mínimo de los valores para los cuales la función de pertenencia es máxima.
