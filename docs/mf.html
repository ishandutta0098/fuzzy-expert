Funciones de pertenencia¶

Las funciones de este módulo devuelven una especificación de función de pertenencia estándar como una lista de puntos (x_i, u_i).

class fuzzy_expert.mf.MembershipFunction(n_points=9)[fuente]¶

Bases: object

Constructor de funciones de pertenencia.

Parámetros

n_points – Número de puntos base para construuir las aproximaciones.

>>> from fuzzy_expert.mf import MembershipFunction
>>> mf = MembershipFunction(n_points=3)
>>> mf(('gaussmf', 5, 1))
[(2, 0), (3.0, 0.1353352832366127), (3.8, 0.48675225595997157), (4.6, 0.9231163463866356), (5.0, 1.0), (5.4, 0.9231163463866356), (6.2, 0.48675225595997157), (7.0, 0.1353352832366127), (8, 0)]
gaussmf(center, sigma)[fuente]¶

Función de pertenencia gaussiana.

Parámetros

  • center – Define el centro de la función de pertenencia.

  • sigma – Define el ancho de la función de pertenencia, donde un valor más grande crea una función de pertenencia más ancha.

>>> from fuzzy_expert.mf import MembershipFunction
>>> mf = MembershipFunction(n_points=3)
>>> mf.gaussmf(center=5, sigma=1)
[(2, 0), (3.0, 0.1353352832366127), (3.8, 0.48675225595997157), (4.6, 0.9231163463866356), (5.0, 1.0), (5.4, 0.9231163463866356), (6.2, 0.48675225595997157), (7.0, 0.1353352832366127), (8, 0)]
gbellmf(center, width, shape)[fuente]¶

Función de pertenencia de campana generalizada.

Parámetros

  • center – Define el centro de la función de pertenencia.

  • width – Define el ancho de la función de pertenencia, donde un valor más grande crea una función de pertenencia más ancha.

  • shape – Define la forma de la curva a cada lado de la meseta central, donde un valor mayor crea una transición más pronunciada.

>>> from fuzzy_expert.mf import MembershipFunction
>>> mf = MembershipFunction(n_points=3)
>>> mf.gbellmf(center=5, width=1, shape=0.5)
[(-1, 0), (0.0, 0.16666666666666666), (1.0, 0.2), (2.0, 0.25), (3.0, 0.3333333333333333), (3.8, 0.45454545454545453), (4.0, 0.5), (4.6, 0.7142857142857141), (5.0, 1.0), (5.4, 0.7142857142857141), (6.0, 0.5), (6.2, 0.45454545454545453), (7.0, 0.3333333333333333), (8.0, 0.25), (9.0, 0.2), (10.0, 0.16666666666666666), (11, 0)]
pimf(left_feet, left_peak, right_peak, right_feet)[fuente]¶

Función de pertenencia en forma de pi.

Parámetros

  • left_feet – Defines the left feet of the membership function.

  • left_peak – Define el pie izquierdo de la función de pertenencia.

  • right_peak – Define el pico derecho de la función de pertenencia.

  • right_feet – Define el pie derecho de la función de pertenencia.

>>> from fuzzy_expert.mf import MembershipFunction
>>> mf = MembershipFunction(n_points=4)
>>> mf.pimf(left_feet=1, left_peak=2, right_peak=3, right_feet=4)
[(1.0, 0.0), (1.3333333333333333, 0.22222222222222213), (1.6666666666666665, 0.7777777777777776), (2.0, 1.0), (3.0, 1.0), (3.3333333333333335, 0.7777777777777776), (3.6666666666666665, 0.22222222222222243), (4.0, 0.0)]
