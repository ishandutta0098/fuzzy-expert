¿Qué es?

Fuzzy-Expert es un paquete de Python para construir Sistemas de Inferencia Borrosa de Zadeh-Mamdani. Fuzzy-Expert admite una combinación de valores difusos y concretos yvalores de certeza asociados. El sistema de inferencia devuelve valores concretos y factores de certeza para las conclusiones de las reglas.

Principales Características

  • Especificación de funciones de pertenencia mediante funciones estándar o como una lista de puntos.

  • Diferentes funciones para especificar T-normas y Conormas.

  • Diferentes operadores de implicación.

  • Composición usando las reglas` max-min` o max-prod.

  • Funciones más comunes para la agregación de producciones»

  • Diferentes operadores para desfuzzificar conjuntos borrosos.
