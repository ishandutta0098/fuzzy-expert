fuzzy_expert.operators.apply_modifiers(membership, modifiers)[fuente]¶

Aplica una lista de modificadores a un vector NumPy.

Parámetros

  • membership – Valores de pertenencia a ser modificados.

  • modifiers – Lista de modificadores.

>>> from fuzzy_expert.operators import apply_modifiers
>>> x = [0.0, 0.25, 0.5, 0.75, 1]
>>> apply_modifiers(x, ('not', 'very'))
array([1.    , 0.9375, 0.75  , 0.4375, 0.    ])

fuzzy_expert.operators.defuzzificate(universe, membership, operator='cog')[fuente]¶

Computa un valor concreto representantivo del conjunto borroso

Parámetros

  • universe – Vector de valores representando el universo de discurso.

  • membership – Vector de valores que representan la función de pertenencia

  • operator – Método usado para calcular el valor concreto representativo de conjunto borroso. * «cog»: Centro de gravedad. * «boa»: Bisector del area. * «mom»: Media de los valores para los cuales la función de pertenencia es máxima. * «lom»: Valor más grande de los valores para los cuales la función de pertenencia es máxima. * «som»: Valor más pequeño de los valores para los cuales la función de pertenencia es máxima.

>>> from fuzzy_expert.operators import defuzzificate
>>> u = [0, 1,   2, 3, 4]
>>> m = [0, 0, 0.5, 1, 1]
>>> defuzzificate(u, m, "cog")
2.9166666666666665

>>> defuzzificate(u, m, "boa")
3.0

>>> defuzzificate(u, m, "mom")
3.5

>>> defuzzificate(u, m, "lom")
4

>>> defuzzificate(u, m, "som")
3
